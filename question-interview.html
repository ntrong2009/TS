Kiểu tĩnh (Static Typing)

Cho phép định nghĩa kiểu cho các biến, hàm, tham số, và giá trị trả về.
  • Phát hiện lỗi sớm: Kiểm tra lỗi tại thời điểm biên dịch (compile-time) thay vì chạy chương trình (run-time).
  • Giúp người khác hiểu mã nguồn dễ dàng hơn và giảm thiểu lỗi liên quan đến kiểu dữ liệu.

Kiểu tổng hợp (Type Inference)

Có khả năng tự động suy luận kiểu dựa trên giá trị gán cho biến hoặc hàm.
  • Giảm bớt việc khai báo kiểu

Kiểu nâng cao (Advanced Types)

TypeScript hỗ trợ các kiểu dữ liệu phức tạp như Union types, Intersection types, và Tuple.
  • Tăng cường tính linh hoạt

Module và Namespace

hỗ trợ hệ thống module, cho phép chia nhỏ mã nguồn thành các phần riêng biệt dễ quản lý.
  • Quản lý dự án lớn dễ dàng hơn

Interface và Abstract Class

TypeScript hỗ trợ định nghĩa interface và lớp trừu tượng.
  • Tăng tính trừu tượng và tái sử dụng mã

Cải thiện công cụ và hỗ trợ IDE

Các IDE như Visual Studio Code cung cấp tính năng hỗ trợ mạnh mẽ cho TypeScript, bao gồm tự động hoàn thành mã, kiểm tra lỗi, và gợi ý.
  • Tăng năng suất lập trình
  • Trải nghiệm lập trình tốt hơn

Khả năng tương thích ngược

TypeScript là siêu ngữ của JavaScript, tức là mọi mã JavaScript hợp lệ đều là mã TypeScript hợp lệ.
  • Dễ dàng chuyển đổi từ JavaScript sang TypeScript: Không cần phải viết lại mã từ đầu, có thể nâng cấp dần dần.
